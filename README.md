# ClearTender

Static landing page for ClearTender, a South African independent quote and tender pricing review service.

## Files
- `index.html` — premium static landing page with lead form
- `README.md` — project notes
- `vercel.json` — static deployment config

## Local use
Open `index.html` in a browser or serve it with any static file server.

## Notes
- Primary CTA: `Request a Tender Review`
- Footer disclaimer: `ClearTender does not provide engineering signoff, QS certification, or legal advice.`
- The lead form opens a prefilled email draft for `estimates@subtleconsulting.co.za`.
